Document Technique du Projet BOBS WorldIntroductionCe document vise à présenter notre solution d'interaction entre des capteurs physiques etun environnement virtuel. Le projet BOBS World met en œuvre une combinaison dematériels et de logiciels pour réaliser cette interaction de manière efficace et intuitive.Matériels et logiciels utilisésMatériels● Arduino Uno : Microcontrôleur pour le traitement et la communication des données.● Capteur flex : Utilisé pour détecter les variations de flexion.● Résistances : Pour ajuster les niveaux de tension.● LEDs : Indicateurs visuels d’état.Logiciels● Arduino IDE : Environnement de développement intégré pour programmer ettéléverser le code sur l'Arduino.● Unreal Engine : Plateforme pour la création et le contrôle de l’environnement virtuelet des objets interactifs.Étapes du projetPhase 1 : Conception électroniqueObjectif : Concevoir et assembler le circuit électronique pour connecter les capteurs flex àl'Arduino.Solution :- Un schéma électrique a été réalisé pour connecter les capteurs flex à l'Arduino Uno. Desrésistances ont été ajoutées pour ajuster les niveaux de tension et garantir des mesuresprécises.- Les LEDs ont été intégrées pour fournir un retour visuel immédiat lors de l'activation descapteurs.- Le code Arduino a été écrit pour lire les valeurs des capteurs et allumer les LEDs lorsquela flexion dépasse un seuil prédéfini.
Extrait de code Arduino :int sensorPin = A0; // Broche analogique pour le capteur flexint ledPin = 13; // Broche pour la LEDint sensorValue = 0;void setup() {pinMode(ledPin, OUTPUT);Serial.begin(9600);}void loop() {sensorValue = analogRead(sensorPin); // Lecture de la valeur du capteurif (sensorValue > 500) { // Seuil de flexiondigitalWrite(ledPin, HIGH); // Allumer la LED} else {digitalWrite(ledPin, LOW); // Éteindre la LED}Serial.println(sensorValue); // Envoyer la valeur au port sériedelay(100); // Délai pour stabiliser la lecture}Résultat : Le circuit fonctionne correctement, avec des LEDs qui s'allument en fonction de laflexion détectée par les capteurs.
